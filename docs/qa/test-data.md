# Test Data

## Demo data
Aplikace inicializuje demo data pri prvnim spusteni v ramci session.

## Doporucene varianty
Vytvor 3 roky a nastav ruzne stavy rozpoctu:
- Rok A: bez rozpoctu
- Rok B: pod limitem (plan < rozpoctu)
- Rok C: nad limitem (plan > rozpoctu)

## Dary
- Koupeno s cenou (validni)
- Koupeno bez ceny (nevalidni)
- Planovano s cenou
- Planovano bez ceny

## Osoby
- Minimalne 3 osoby s ruznou historii
- Jedna osoba bez darku (test prazdneho stavu)
